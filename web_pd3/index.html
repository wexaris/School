Task #2 - Dictionary

Use the "my_dict" object (loaded from dictionary.js) to implement a simple dictionary on this page.

Note: you may modify dictionary.js if you wish.

1) Word lookup

  • Load the list of "my_dict" dictionary keys (English words) into the dropdown list below. The list must be sorted!!! (A..Z)
  • When the user chooses a word from the dropdown list, display the Latvian translation of this word (as a part of the webpage).

2) Translation tool

  • Add a text input field and a button ("Translate") to this page.
  • When the user presses "Translate", translate contents of the input field by replacing English words in the dictionary with their Latvian translations. Display the translation below the input field.
    • Word replacement should be case insensitive (e.g. both "Dog" and "dog" should be translated)
    • Unrecognized word should remain unchanged
  • Add 2 translation examples to the "Example:" section (you may delete the existing content).

Input field + button + translation results display goes here!

Task #3

Multiplication table - create (here or in a separate HTML page):

  • 2 text input fields (A and B) that accept integer values from 1 to 25
    • validation of input values - when the user exits the entry field, display a warning message (next to the entry field) in case the value is not in the permitted range
  • two buttons: "Table" and "Clear"
    • the "Table" button can be pressed only if both entry field values are valid. when pressed it should display a multiplication table for values (1..A) x (1..B).
    • the "Clear" button should hide (or delete) the table

It is up to you how to display the multiplication table on the page.
